# YouTube Content Strategist

This plugin contains a suite of skills and tools for researching, ideating, and planning YouTube videos. The skills are designed to work together to create a cohesive workflow for optimizing YouTube content performance.

## Requirements

1. **YouTube Data API Key**: You need a YouTube Data API key to access the YouTube Analytics tools. You can create a key by following the instructions [here](https://developers.google.com/youtube/v3/getting-started).
2. **GEMINI API KEY**: You need a Gemini API key to use the NanoBanana image generation model. You can create a key by following the instructions [here](https://makersuite.google.com/app/apikey).
2. **Thumbkit**: This plugin uses Thumbkit, a CLI tool for generating and editing high-performing YouTube thumbnails. Thumbkit is built on top of Gemini 2.5 Flash (NanoBanana) image generation model. You can install Thumbkit by following the instructions [here](https://github.com/kenneth-liao/thumbkit).
3. **uv**: This plugin uses [uv](https://docs.astral.sh/uv/) to manage python projects. You can install uv by following the instructions [here](https://docs.astral.sh/uv/getting-started/installation/).

## Getting Started

1. If you haven't already, install the project by following the `Getting Started` instructions in the project root [README](../../../README.md).
2. Once the `ai-launchpad` project is installed, navigate to the `ai_launchpad/claude_code_module/ai-launchpad-marketplace/yt-content-strategist` directory and start running the code in any file.
